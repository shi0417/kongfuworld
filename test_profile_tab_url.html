3. URL参数格式说明
Profile页面支持的URL参数格式
支持的URL格式：
• http://localhost:3000/profile - 默认显示Edit Profile选项卡
• http://localhost:3000/profile?tab=profile - 显示Edit Profile选项卡
• http://localhost:3000/profile?tab=notifications - 显示Notifications选项卡
• http://localhost:3000/profile?tab=settings - 显示Settings选项卡
实现原理：
1. 使用 useLocation() hook获取当前URL
2. 使用 URLSearchParams 解析查询参数
3. 根据 tab 参数设置对应的选项卡状态
4. 当URL变化时自动更新选项卡显示
4. 常见问题排查
如果URL参数处理不工作，可能的原因和解决方案
可能的问题：
1. React Router版本问题： 确保使用正确的React Router版本
2. useLocation hook未导入： 检查是否正确导入了useLocation
3. useEffect依赖项： 确保useEffect的依赖项包含location.search
4. 选项卡状态初始化： 确保选项卡状态正确初始化
5. 浏览器缓存： 清除浏览器缓存后重新测试
